# Rease Bot 🤖

Многофункциональный Telegram бот для групповых чатов с играми, управлением чатом и самообучением.

## Технологии

- Python 3.11
- aiogram 3
- SQLAlchemy + aiosqlite
- pydantic-settings

## Установка и запуск

1. Клонировать репозиторий

git clone https://github.com/rearzty/reasebot

2. Установить зависимости

pip install -r requirements.txt

3. Создать .env файл по примеру .env.example

BOT_TOKEN=токен_от_BotFather
WEATHER_API_KEY=ключ_от_openweathermap
DATABASE_URL=sqlite+aiosqlite:///bot.db

4. Запустить

python main.py

## Команды

### 🎮 Игры
/startmafia — запустить игру в мафию
/guessnumber — угадай число от 1 до 100
/rps — камень ножницы бумага с другим игроком

### 💑 Социальные
/marry — предложить брак пользователю
/divorce — развестись
/getmarried — список пар в чате
/kiss — поцеловать пользователя
/hug — обнять пользователя

### 🛠 Управление чатом
/mute — заглушить пользователя
/unmute — разглушить пользователя
/mutedlist — список заглушенных
/kick — кикнуть пользователя

### ℹ️ Информация
/getinfo — информация о пользователе
/chatinfo — информация о чате
/weather — погода в городе

## Архитектура

handlers/ — обработчики команд
database/ — модели и запросы к БД
middlewares/ — проверка мута
services/ — внешние API (погода)

## Особенности

- Асинхронный — работает на aiogram 3
- Самообучение — бот запоминает ответы пользователей
- Мафия с полноценными ролями — Дон, Доктор, Комиссар, Мафия, Мирный
- Система браков между пользователями
